Bienvenido a ANGIE

Manual de Instalación y Usuario

El presente manual tiene como objetivo guiar al usuario final en el proceso de instalación de la aplicación “ANGIE: AGENDA INTELIGENTE” en su dispositivo Android, así como proporcionar una guía clara y detallada sobre el uso de todas sus funciones. Al finalizar la lectura, el usuario podrá gestionar su cuenta, agendar eventos, modificarlos y administrar su agenda de manera eficiente.

Manual de Instalación

La aplicación se distribuye mediante un archivo APK (Android Package Kit). Siga estos pasos para instalarla correctamente en su dispositivo.

Requisitos previos

  • Dispositivo con sistema operativo Android versión 5.0 o superior.
  • Permitir la instalación de aplicaciones de orígenes desconocidos.

Pasos de instalación

1

Transferir el archivo APK

Copie el archivo ANGIE.apk a la memoria interna de su dispositivo o a una tarjeta SD.

Nota: Este paso puede omitirse, debido a que el proceso de instalación difiere en Android 8.0 o superior. Solo basta con localizar el APK y activar la instalación.
Versión de Android necesaria para ANGIE
2

Habilitar orígenes desconocidos

Vaya a Ajustes → Seguridad (o Privacidad). Active la opción "Permitir instalación de aplicaciones de orígenes desconocidos".

Nota: En Android 8.0 o superior, esta opción puede aparecer al momento de abrir el APK, esta situación se reemplaza por: ‘Abrir Archivo’.
Permitir instalación desde orígenes desconocidos
3

Ejecutar el instalador

Abra el administrador de archivos de su dispositivo. Localice el archivo ANGIE.apk y presiónelo. Confirme la instalación presionando "Instalar".

Presiona "Instalar" para continuar
4

Finalizar

Una vez completada la instalación, presione "Abrir" para iniciar la aplicación o "Listo" para salir.

Manual de Usuario

1. Acceso al sistema

Al abrir la aplicación, se mostrará la pantalla "¿Cómo deseas acceder?" con dos opciones:

  • Crear una nueva cuenta (para usuarios nuevos).
  • Iniciar sesión (para usuarios ya registrados).
Pantalla principal de acceso a ANGIE

1.1 Crear una nueva cuenta

Seleccione esta opción si es la primera vez que usa la aplicación. Complete todos los campos obligatorios:

  • Nombre de usuario (mínimo 4 dígitos)
  • Nombre
  • Apellido Paterno
  • Apellido Materno
  • E-mail (debe tener formato válido, ej: usuario@dominio.com)
  • Contraseña (mínimo 8 caracteres: al menos una mayúscula, una minúscula y un número)

Presione el botón "Continuar". Verá el mensaje: "Cuenta creada exitosamente". Presione el botón Atrás para regresar a la pantalla de acceso.

Formulario para crear nueva cuenta

1.2 Iniciar sesión (cuenta ya existente)

Seleccione la opción "Iniciar Sesión". Ingrese su Usuario y Contraseña. Presione "Confirmar". Si los datos son correctos, accederá a la pantalla principal (Home).

Credenciales de acceso a ANGIE

1.3 Restablecimiento de contraseña (¿Olvidó su contraseña?)

En la pantalla de Iniciar Sesión, presione el enlace "¿No recuerdo mi contraseña?". Ingrese su Usuario y Correo Electrónico registrados. Presione "Continuar".

El sistema mostrará un código de verificación. Escríbalo exactamente como se muestra. Presione "Continuar". Ahora cree una nueva contraseña (cumpliendo las reglas de seguridad) y confírmela. Presione "Confirmar". Será redirigido a la pantalla de Inicio de Sesión. Ingrese con su usuario y la nueva contraseña.

Validaciones importantes

Campo Requisito
Usuario Mínimo 4 dígitos
Correo Formato válido (nombre@dominio.com)
Contraseña Mínimo 8 caracteres, una mayúscula, una minúscula, un número

2. Pantalla principal (Home)

Una vez dentro de la aplicación, verá un calendario.

Seleccionar fecha: Haga doble clic (en un lapso de 400 milisegundos) sobre el día deseado para agendar un evento.

Nota importante: Solo se pueden seleccionar fechas vigentes (hoy o futuras). Las fechas pasadas aparecerán inhabilitadas.
Vista del calendario en pantalla principal

3. Agendar evento

Al seleccionar una fecha válida, se abrirá el formulario de registro.

3.1 Datos del evento

Campo Obligatorio Validación
Usuario Sí Debe existir en el sistema (el mismo con el que creó la cuenta)
Nombre del evento Sí Mínimo 3 caracteres
Descripción No Campo opcional

Presione "Siguiente" para continuar.

Completar detalles del evento

3.2 Agendar hora del evento

Seleccione una hora de la lista desplegable (intervalos sugeridos). Si la hora que necesita no aparece, presione "Especificar hora manualmente" y elija la hora exacta. Presione "Confirmar" para guardar el evento. Será regresado automáticamente a la pantalla.

Selección de hora manual o automática

4. Ver eventos

Acceda a esta sección desde el menú principal. Se mostrarán todos sus eventos registrados con fecha y hora.

Nota: Los eventos con fechas ya pasadas se eliminarán automáticamente del sistema.
Visualización de eventos agendados

5. Modificar eventos

Vaya a la sección "Modificar Eventos". Seleccione el evento que desea editar. Puede cambiar: Fecha, Nombre del evento, Descripción, Tipo de actividad, Hora. Presione "Guardar cambios". También puede eliminar cualquier evento de forma manual presionando el botón "Eliminar".

Edición o eliminación de eventos

6. Perfil de usuario

En esta sección podrá visualizar sus datos personales registrados: Nombre de usuario, Nombre completo (Nombre, Apellido Paterno, Apellido Materno), Correo electrónico.

Nota: Por seguridad, los datos del perfil no se pueden modificar desde esta sección. Para cambiarlos, deberá crear una nueva cuenta.
Información del perfil visible
